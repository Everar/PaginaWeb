CLORIN-HEK

 

Manual
En este manual se dará a conocer el uso de la aplicación Clorin hek la cual es un prototipo que busca encontrar metales pesados en un recipiente

Arduino

 

La utilización de un arduino uno, El cual es el intermediario entre los sensores y la conectividad con una aplicación mediante Bluetooth




Colocación de sensor


Se determinan por la ubicación en el cual se determinan tres valores que son representados por medio de cables uno color marrón dos color negro y tres color azul, "El color marron " (VCC) "El color azul" (GND), "El color negro" (Digital)




Utilización del Foco Led

El led es una manera de representar de forma física la funcionalidad del sensor inductivo



Bluetooth H6


El medio por el cual existe la conectividad con la aplicación móvil donde se registran los datos obtenidos del sensor




Aplicación


En el desarrollo de una aplicación que facilite el manejo para el usuario en el cual tenga una mayor experiencia y pueda disfrutar de los beneficios que proporciona la aplicación móvil
